Welcome to shehacks.it

May 20, 2026 · welcome, intro

This is the first post on shehacks.it — a place where I’ll be writing about vulnerability research, offensive security concepts, and how AI is changing the way we approach penetration testing.

What to expect

Posts here will fall into a few categories:

  • Vulnerability Breakdowns — When interesting CVEs drop, I’ll dig into what they are, why they matter, and how to test for them.
  • Intro to Concepts — Accessible walkthroughs of offensive security topics for people getting started or looking to sharpen specific skills.
  • AI-Assisted Testing — Practical exploration of how AI tools can augment (not replace) the pentester’s workflow.

How this site works

This blog runs on Jekyll and GitHub Pages. Every post is a markdown file in the _posts folder. To write a new post, I create a file like 2026-05-20-my-post-title.md, add the front matter at the top, and push to GitHub. That’s it — no CMS, no database, no hosting fees.

Stay tuned.
