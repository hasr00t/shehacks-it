Welcome to shehacks.it

May 20, 2026 · welcome, intro

First post. This site is where I’ll write about vulnerability research, offensive security concepts, and using AI in penetration testing.

What goes here

Posts will generally fall into a few buckets:

  • Vulnerability Research - Digging into interesting CVEs as they come out. What they are, why they matter, how to test for them.
  • Intro to Concepts - Walkthroughs of offensive security topics, aimed at people getting started or brushing up on specific areas.
  • AI-Assisted Testing - Practical looks at how AI tools fit into the pentesting workflow.

How this site works

Jekyll and GitHub Pages. Every post is a markdown file in the _posts folder. New post goes in, push to GitHub, site rebuilds. No CMS, no database, no hosting costs.
